# teemi CAD workflows

This directory now has two clear purposes:

- `inspiration_notebooks/`
  Curated example workflows maintained with the repo.
- `SynBio_27460_examples/`
  A home for student project notebooks and course-specific work.

The goal is to keep the inspiration material reusable and stable, while giving students a separate space to add their own projects without mixing them into the curated examples.
