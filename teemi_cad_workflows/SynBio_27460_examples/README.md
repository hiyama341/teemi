# SynBio_27460 examples

This folder is for student project notebooks and supporting files for the `SynBio_27460` course.

Suggested structure:

- one folder or notebook per project
- a short README for each project if multiple files are needed
- project-specific data kept close to the notebook when possible

Guidelines for students:

- use clear notebook names
- explain the biological question at the top
- document required input files and assumptions
- keep examples reproducible enough that others on the course can follow them
