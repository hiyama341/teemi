# Inspiration notebooks

This folder contains the curated `teemi` workflow examples.

Structure:

- `notebooks/`
  Reusable inspiration notebooks.
- `data/`
  Data files used by the inspiration notebooks.

Guidelines:

- Keep these notebooks broadly reusable.
- Keep the biological goal clear and well explained.
- Store notebook-specific files in `data/`.
- Prefer lightweight, well-scoped examples over very large project dumps.
