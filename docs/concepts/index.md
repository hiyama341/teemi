# Concepts

`teemi` is built around practical, reproducible DBTL workflows for bioengineering. The main ideas are:

- notebook-first scientific programming
- reusable workflow building blocks for design, build, test, and learn stages
- maintainable experiment context through lightweight LIMS integrations
