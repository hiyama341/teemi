Workflow Gallery
================

The workflow gallery highlights the lighter, workflow-oriented notebooks in
``teemi_cad_workflows/inspiration_notebooks/notebooks``. These examples are a good
fit when you want a focused starting point rather than a long end-to-end case study.

The current direction of the repository is to expand this gallery while keeping the
larger strictosidine notebooks available under :doc:`colab_notebooks`.
